## 🔧 完整配置 (可选)

如果您想使用完整功能：

### 1. 创建 Supabase 项目
1. 访问 [supabase.com](https://supabase.com)
2. 创建新项目
3. 在 SQL 编辑器中执行 `database-schema.sql`

### 2. 配置环境变量
编辑 `.env` 文件：
```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

### 3. 重启服务器
```bash
# 停止服务器 (Ctrl+C)
npm run dev
```
